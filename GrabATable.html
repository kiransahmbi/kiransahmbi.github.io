Grab A Table Web App

A Web Application that allows users to book a reservation at a desired restaurant.
Vendors are able to view a list of reservations and a dashboard depicting reservations by date.

Requirements

Customer User Stories

As a Customer...
I can view restaurants listed, organized in different categories (cuisine, trending), so that I can view the options available.
I can view restaurant photos, layout, menu, hours of service, capacity, and location, so that I can make a well-informed decision.
I can reserve a table at my selected restaurant for a specified time frame, so that I can reserve a space for my guests.
I should receive confirmation of my reservation so that if I choose, I can cancel my reservation at a later time.

Vendor User Stories

As a Vendor...
I can make changes to my restaurant information so that my restaurant information is up to date.
I can view a list of current reservations to prepare for business operations.
I can modify table reservations so that I can avoid or handle scheduling conflicts.
