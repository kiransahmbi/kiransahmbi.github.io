Sell Used Products Web App

A Web Application that allows authenticated users to post listings of products they wish to sell.
All listings and details can be viewed publically, where anonymous users may post questions on a particular listing.

Requirements

  1. User Management and Site Security
    • User Registration allows user to enter profile information (username, password, email address, etc.) to be stored in MongoDB database.
    • User will be able to Login, Logout (JWT Authentication), and view/edit their posted listings.
    • Non-registered users cannot create, edit, delete listings or enter secure areas of the site.

  2. Listings
    • After a user is registered and logged in, they can post public advertisement (CRUD).
    • User can specify Listing Start Date and Lifetime of the advertisement (i.e. when listing becomes active/enabled, when it expires).
    • A registered user will be able to edit, delete any advertisement that they owns (CRUD).

  3. Questions Feature
    • Anonymous users can post public questions on any active listing.
    • Only the user that published the ad can post an answer for a question.

Functionality

Product Listings (As Unauthenticated User)

Public Product Listings
Product Details Page
Ask an Anonymous Question

Authenticated User Features

Sign In
Product Page as Authenticated User
Update Product Listing
Answer Question on Listing
