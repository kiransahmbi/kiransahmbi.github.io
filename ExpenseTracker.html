Expense Tracker App

A C# desktop application that tracks your expenses!
Features DataViewGrid CRUD, and LINQ Queries

Github Repository

Requirements

  1. Data Persistence (through static ExpenseManager Class)
    • Defines CRUD Methods that serialize/deserialize data from XML file

  2. Expense Class
    • Defines Expense Object: ID, Vendor, Expense Date, Category, Name, Cost

  3. LINQ Queries
    • Ability to filter by Expense Category, Start/End Date
    • Group Queries by Category or Vendor to calculate Total Cost, Average Cost, and Min/Max

  4. Error Handling
    • Program ensures all fields are filled with valid inputs before CRUD actions are initiated

  5. Status Bar
    • LinkedList is utilized to track all actions made during runtime session. Actions displayed in status bar

  6. User Interface (XAML)
    • Utilizes Grid and StackPanels to organize components
    • Components: Grid, TabControl, TextBox, RadioButton, CheckBox, Expander
